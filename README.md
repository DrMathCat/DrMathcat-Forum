## How to participate?

Click the [**Discussions** ](https://github.com/DrMathCat/DrMathcat-Forum/discussions)tab above and choose the relevant 
category. A free GitHub account is required to post.


## Subscribe to a Specific Post

If you want to follow a particular discussion thread, open the 
post and click the **Subscribe** button in the right sidebar. 
You will receive a notification whenever someone replies to that thread.

## Subscribe to the Entire Forum

If you want to be notified every time a new post is created:

1. Go to the [main page of this repository](https://github.com/DrMathCat/DrMathcat-Forum)
2. Click the **Watch** dropdown in the top right corner
3. Select **Custom**
4. Check **Discussions**
5. Click **Apply**

You will now receive notifications for all new discussions and replies.

## Where Will I Receive Notifications?

By default, notifications appear in your GitHub inbox (the bell icon). 
To also receive email notifications, go to  **profile avatar**(top right corner)->**Settings → Notifications** 
and enable email under **Participating and watching**.

## Writing Math Formulas

**Inline formula** — use single `$`:

`$f'(x) = 2x$` renders as $f'(x) = 2x$

**Display formula** — use double `$$`:

```
$$\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$
```
